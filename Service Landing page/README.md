# Digital Chitra Landing Page

Next.js and Tailwind CSS landing page for booking a free 1:1 digital marketing consultation call.

## Run locally

```bash
npm run dev
```

Open `http://localhost:3000`.

## Important content placeholders

Update [lib/site-config.ts](/D:/Digital%20Marketing/Lead%20magnet/Service%20Landing%20page/lib/site-config.ts) when you are ready to add:

- Vimeo video link for the `/thanks` page
- WhatsApp chat link for the `/thanks` page

## Flodesk setup

The visible booking form uses the website design, and the backend posts the same data to your Flodesk form submission endpoint so Flodesk capture and automation can still run without changing the page layout.
